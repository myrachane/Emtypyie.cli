# EMTYPYIE.CLI

> Run emtypyie projects from your terminal with ease — the future runtime engine.

## Structure

| Path | Description |
|------|-------------|
| `root4node/` | Node.js CLI — published to npm as `emtypyie-cli` |
| `root4c/`    | C CLI — portable single-binary port (in development) |
| `mainsite/`  | Website landing pages |
| `manifests/` | Winget package manifests |
| `choco/`     | Chocolatey package |
| `dev/`       | CDN static JSON assets |

## Quick links

- **Node CLI:** [root4node/](root4node/) — `npm install -g emtypyie-cli`
- **C CLI (WIP):** [root4c/](root4c/)
- **Website:** [emtypyie.in/cli](https://emtypyie.in/cli)
- **Winget:** `winget install myrachane.emtypyie-cli`
